# Dotfiles
These dotfiles are managed by : **Gnu Stow**. They say it's a sane
configuration.


## Overview

- OS : Linux
- Distro : Arch
- WM : Sway
- Code Editor : Neovim 
- Terminal : Foot

## Instructions
_Clone the repo_ : 
`$ git clone https://github.com/Yushi5058/dotfiles.git`

_Use restow-all zsh function_: 
`$ restow-all`

You're good to go!
